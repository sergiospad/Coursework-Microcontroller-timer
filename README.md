Устройство представляет собой электронный таймер с возможностью установки временного интервала, визуальной индикацией обратного отсчёта и сохранением настроек в энергонезависимой памяти. 
Управление осуществляется тремя кнопками. Информация о текущем времени отображается на семисегментном индикаторе с точностью до десятой доли секунды. По окончании заданного интервала активируется светодиодный индикатор.
